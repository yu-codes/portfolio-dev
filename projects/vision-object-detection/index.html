Vision Object Detection

Deep learning-based computer vision system for real-time object detection and classification, featuring YOLO models, custom training pipelines, and production-ready deployment via TorchScript/ONNX.

Python PyTorch Computer Vision Deep Learning YOLO

Overview

A production-grade computer vision system leveraging state-of-the-art deep learning models for robust object detection and classification. The project implements end-to-end pipelines — from data preprocessing and model training through inference optimization to deployment — targeting real-world applications such as facility inspection and anomaly detection.

Key Features

  • Real-time Object Detection: YOLO v5/v8 implementation achieving 85 FPS on RTX 3090 with 11.8ms latency
  • Flexible Model Architecture: Configurable backbones (ResNet, EfficientNet) with anchor-based and anchor-free detection heads
  • Advanced Augmentation: Comprehensive pipeline including rotation, flip, color jittering, mosaic, and mixup augmentation
  • Transfer Learning: Pre-trained on COCO dataset with fine-tuning support for custom domains
  • Production Export: One-command export to TorchScript and ONNX formats for edge deployment
  • Confidence Filtering & NMS: Configurable post-processing with class-specific confidence thresholds
  • Multi-format Input: Support for images, video files, and real-time camera streams
