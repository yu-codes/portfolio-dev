Overview

Agent-Arena (b-box) ‚Äî Black-Box for Boxing LLM Behaviors ‚Äî is an AI Agent evaluation platform designed to assess, quantify, and compare the behavioral performance of different LLM agents. It features a plugin-based architecture for extensibility, an async evaluation engine for parallel metric computation, and a modern Vue 3 dashboard with radar chart visualization.

Key Features

  • Multi-Agent Support: OpenAI / Anthropic / Custom API with extensible Adapter pattern ‚Äî add new agents by implementing BaseAgent and registering
  • 12 Universal Metrics: Semantic similarity, factual accuracy, hallucination detection, coherence, safety, toxicity, response completeness, and more
  • 5 Custom Metrics: Dynamically provided per agent type ‚Äî tool usage scoring, schema validation, exact match, format compliance, API call analysis
  • Plugin Architecture: New agents and metrics require zero modification to core code ‚Äî just implement, register, and deploy
  • Async Evaluation Engine: All metrics execute in parallel using Python asyncio for maximum throughput
  • Report Generation: Detailed evaluation reports exportable in JSON and PDF formats
  • Modern Dashboard: Vue 3 SPA with radar chart visualization, score breakdowns, and real-time evaluation status
  • Containerized: Full stack runs with a single docker compose up --build command
