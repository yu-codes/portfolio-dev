Overview

A comprehensive technical platform combining personal portfolio, learning hub, and project showcase. Built entirely with Hugo static site generation, featuring a sophisticated interactive roadmap visualization system inspired by XMind. Supports dual-language content (English & Traditional Chinese) and deploys seamlessly to GitHub Pages with zero backend requirements.

Key Features

  • XMind-Style Interactive Roadmap: 660+ line SVG-based collapsible node system with auto-layout algorithm, smooth animations, and context-aware link rendering for navigating 100+ articles across 9 domains
  • Dual-Repo Deployment: GitHub Actions workflow automatically deploys develop → portfolio-dev and main → portfolio for staging/production separation
  • Full Bilingual Support: English & Traditional Chinese with separate content directories, language toggle, and auto-generated navigation
  • Sophisticated Dark Mode: Persistent theme preference with system detection fallback, CSS variable–driven theming across all components
  • Modular CSS Architecture: Design token system in base.css, structural layout in layout.css, reusable components in components.css, and page-specific overrides in pages/
  • Mobile-First Responsive Design: Adaptive layouts with CSS Grid and Flexbox for all breakpoints
  • SEO Optimized: Auto-generated multi-language sitemaps, structured metadata, semantic HTML, and Open Graph tags
