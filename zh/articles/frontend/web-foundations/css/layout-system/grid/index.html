CSS Grid Layout 是強大的二維佈局系統，能同時控制行（row）與列（column）的排列。它從根本上改變了網頁佈局的方式，讓複雜的頁面結構變得直觀且易於維護。

核心概念

基本術語

  • Grid Container：設置 display: grid 的元素
  • Grid Item：容器的直接子元素
  • Grid Line：分隔行列的線，從 1 開始編號
  • Grid Track：兩條相鄰線之間的空間（行或列）
  • Grid Cell：一行一列的交叉區域
  • Grid Area：由多個 Cell 組成的矩形區域

定義網格結構

.container {
  display: grid;
  grid-template-columns: 200px 1fr 1fr;  /* 三欄 */
  grid-template-rows: auto 1fr auto;      /* 三行 */
  gap: 1rem;                               /* 間距 */
}

fr 單位

fr（fraction）是 Grid 專屬的彈性單位，代表可用空間的比例份額：

/* 三等分 */
grid-template-columns: 1fr 1fr 1fr;

/* 左側固定，右側彈性 */
grid-template-columns: 250px 1fr;

/* 1:2:1 比例 */
grid-template-columns: 1fr 2fr 1fr;

repeat() 函數

/* 12 欄等寬網格 */
grid-template-columns: repeat(12, 1fr);

/* 自動填充，最小 250px */
grid-template-columns: repeat(auto-fill, minmax(250px, 1fr));

/* 自動適應，最小 250px */
grid-template-columns: repeat(auto-fit, minmax(250px, 1fr));

架構總覽

命名區域佈局（Grid Template Areas）

.layout {
  display: grid;
  grid-template-areas:
    "header  header  header"
    "sidebar content aside"
    "footer  footer  footer";
  grid-template-columns: 200px 1fr 200px;
  grid-template-rows: auto 1fr auto;
  min-height: 100vh;
}

.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }
.content { grid-area: content; }
.aside   { grid-area: aside; }
.footer  { grid-area: footer; }

/* 響應式調整 */
@media (max-width: 768px) {
  .layout {
    grid-template-areas:
      "header"
      "content"
      "sidebar"
      "aside"
      "footer";
    grid-template-columns: 1fr;
  }
}

項目定位

/* 以線號定位 */
.item {
  grid-column: 1 / 3;    /* 第 1 線到第 3 線（跨兩欄） */
  grid-row: 2 / 4;       /* 第 2 線到第 4 線（跨兩行） */
}

/* span 關鍵字 */
.item {
  grid-column: span 2;   /* 佔兩欄 */
  grid-row: span 3;      /* 佔三行 */
}

/* 命名線 */
.container {
  grid-template-columns: [sidebar-start] 200px [sidebar-end content-start] 1fr [content-end];
}

.sidebar {
  grid-column: sidebar-start / sidebar-end;
}

對齊系統

.container {
  /* 軌道在容器中的對齊 */
  justify-content: center;  /* 水平方向 */
  align-content: center;    /* 垂直方向 */

  /* 項目在 Cell 中的對齊 */
  justify-items: stretch;   /* 水平方向（預設） */
  align-items: stretch;     /* 垂直方向（預設） */
}

.item {
  /* 個別項目對齊 */
  justify-self: end;
  align-self: center;
}

隱式網格

當項目超出定義的網格範圍時，自動建立隱式軌道：

.container {
  grid-auto-rows: minmax(100px, auto);  /* 隱式行的最小高度 */
  grid-auto-flow: dense;                 /* 密集填充演算法 */
}

關鍵比較

auto-fill vs auto-fit

特性auto-fillauto-fit
行為盡可能填入多軌道填入後坍塌空軌道
空間不足時相同相同
空間充裕時保留空軌道項目擴展填滿
適用場景固定網格結構自適應卡片佈局
/* auto-fill：保留空軌道空間 */
grid-template-columns: repeat(auto-fill, minmax(200px, 1fr));

/* auto-fit：項目擴展填滿所有空間 */
grid-template-columns: repeat(auto-fit, minmax(200px, 1fr));

Grid vs Flexbox 選擇指南

需求建議
一維排列（單行/單列）Flexbox
二維佈局（行＋列）Grid
內容決定大小Flexbox
佈局決定大小Grid
元件內部對齊Flexbox
頁面級結構Grid
動態數量的項目兩者皆可
項目重疊Grid

minmax() vs clamp()

/* minmax：用於 Grid 軌道定義 */
grid-template-columns: minmax(200px, 1fr);

/* clamp：用於一般 CSS 屬性 */
width: clamp(200px, 50%, 800px);

應用場景

場景一：響應式卡片網格

無需媒體查詢即可自適應：

.card-grid {
  display: grid;
  grid-template-columns: repeat(auto-fit, minmax(280px, 1fr));
  gap: 1.5rem;
  padding: 1rem;
}

場景二：Dashboard 佈局

不同大小的面板組合：

.dashboard {
  display: grid;
  grid-template-columns: repeat(4, 1fr);
  grid-auto-rows: minmax(150px, auto);
  gap: 1rem;
}

.widget-large {
  grid-column: span 2;
  grid-row: span 2;
}

.widget-wide {
  grid-column: span 2;
}

場景三：雜誌式圖文排版

圖片跨多行、文字環繞的複雜佈局：

.magazine {
  display: grid;
  grid-template-columns: repeat(6, 1fr);
  grid-auto-rows: minmax(50px, auto);
  gap: 1rem;
}

.featured-image {
  grid-column: 1 / 4;
  grid-row: 1 / 4;
}

.article-text {
  grid-column: 4 / 7;
  grid-row: 1 / 5;
}

場景四：表單雙欄佈局

Label 與 Input 對齊的表單：

.form-grid {
  display: grid;
  grid-template-columns: auto 1fr;
  gap: 0.75rem 1rem;
  align-items: center;
}

.full-width {
  grid-column: 1 / -1;
}
