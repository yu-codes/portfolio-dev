Flexbox 彈性佈局

Flexbox（Flexible Box Layout）是 CSS3 引入的一維佈局模型，專為在容器內分配空間與對齊項目而設計。它解決了傳統 float 與 inline-block 佈局的諸多痛點，成為現代網頁佈局的核心工具。

核心概念

雙軸系統

Flexbox 基於兩條軸線運作：

  • 主軸（Main Axis）：由 flex-direction 決定方向，預設為水平（row）
  • 交叉軸（Cross Axis）：垂直於主軸的方向

容器與項目

Flexbox 分為兩個層級的屬性：

容器屬性（Parent）：

  • display: flex | inline-flex
  • flex-direction: row | row-reverse | column | column-reverse
  • flex-wrap: nowrap | wrap | wrap-reverse
  • justify-content: 主軸對齊
  • align-items: 交叉軸對齊（單行）
  • align-content: 交叉軸對齊（多行）
  • gap: 項目間距

項目屬性（Children）：

  • flex-grow: 擴展比例（預設 0）
  • flex-shrink: 收縮比例（預設 1）
  • flex-basis: 初始大小（預設 auto）
  • align-self: 個別項目的交叉軸對齊
  • order: 排列順序

flex 縮寫屬性

/* flex: grow shrink basis */
.item { flex: 1 1 0; }      /* 等比填滿 */
.item { flex: 0 0 200px; }  /* 固定寬度，不伸縮 */
.item { flex: 1 0 auto; }   /* 可擴展，不收縮 */
.item { flex: none; }       /* 等同 0 0 auto */
.item { flex: auto; }       /* 等同 1 1 auto */

架構總覽

常見佈局模式

置中對齊（水平 + 垂直）

.container {
  display: flex;
  justify-content: center;
  align-items: center;
  min-height: 100vh;
}

導覽列佈局

.navbar {
  display: flex;
  justify-content: space-between;
  align-items: center;
  padding: 0 1rem;
}

.navbar-links {
  display: flex;
  gap: 1rem;
}

卡片等高佈局

.card-container {
  display: flex;
  flex-wrap: wrap;
  gap: 1.5rem;
}

.card {
  flex: 1 1 300px; /* 最小 300px，可擴展 */
  display: flex;
  flex-direction: column;
}

.card-body {
  flex: 1; /* 填滿剩餘空間，實現等高 */
}

.card-footer {
  margin-top: auto; /* 推到底部 */
}

Holy Grail 佈局

.layout {
  display: flex;
  flex-wrap: wrap;
  min-height: 100vh;
}

.header, .footer {
  flex: 1 1 100%;
}

.sidebar-left {
  flex: 0 0 200px;
}

.main {
  flex: 1;
  min-width: 0; /* 防止內容溢出 */
}

.sidebar-right {
  flex: 0 0 250px;
}

進階技巧

處理溢出問題

.flex-item {
  min-width: 0;    /* 允許文字截斷 */
  overflow: hidden;
  text-overflow: ellipsis;
  white-space: nowrap;
}

最後一行左對齊

.grid-like {
  display: flex;
  flex-wrap: wrap;
  gap: 1rem;
}

.grid-like::after {
  content: '';
  flex: auto; /* 填充最後一行的剩餘空間 */
}

關鍵比較

Flexbox vs Float

面向FlexboxFloat
垂直置中一行 CSS需 hack
等高欄位自動不支援
順序控制order 屬性需改 HTML
間距控制gap 屬性margin hack
響應式天生彈性需 clearfix

Flexbox vs Grid

面向FlexboxGrid
維度一維（行或列）二維（行與列）
對齊基準內容驅動容器驅動
適用場景元件內部佈局頁面整體佈局
項目大小由內容決定由軌道定義
重疊控制不支援支援（grid-area）

justify-content 各值比較

值效果
flex-start靠主軸起點
flex-end靠主軸終點
center居中
space-between兩端對齊，項目間等距
space-around每個項目兩側等距
space-evenly所有間隔完全相等

應用場景

場景一：響應式導覽列

小螢幕時切換為垂直排列，大螢幕水平展開：

.nav {
  display: flex;
  flex-direction: column;
}

@media (min-width: 768px) {
  .nav {
    flex-direction: row;
    justify-content: space-between;
  }
}

場景二：表單佈局

Label 與 Input 的彈性排列，可隨螢幕寬度調整：

.form-row {
  display: flex;
  flex-wrap: wrap;
  gap: 0.5rem;
  align-items: center;
}

.form-row label {
  flex: 0 0 120px;
}

.form-row input {
  flex: 1;
  min-width: 200px;
}

場景三：媒體物件（Media Object）

圖片固定寬度，文字區域填滿剩餘空間：

.media {
  display: flex;
  gap: 1rem;
  align-items: flex-start;
}

.media-image {
  flex-shrink: 0;
  width: 80px;
}

.media-body {
  flex: 1;
}

場景四：Sticky Footer

頁尾始終貼底，即使內容不足一螢幕：

body {
  display: flex;
  flex-direction: column;
  min-height: 100vh;
}

main {
  flex: 1;
}
