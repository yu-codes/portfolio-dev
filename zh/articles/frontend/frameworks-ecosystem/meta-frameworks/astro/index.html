Astro

Astro 是一個以內容為導向的 Web 框架，預設輸出零 JavaScript 的靜態 HTML，僅在需要互動的「島嶼」（Islands）中載入 JS。其獨特的多框架整合能力允許在同一專案中混用 React、Vue、Svelte 等元件，為內容豐富的網站提供極致效能。

應用場景

文件網站

Astro 的 Starlight 模板專為文件網站設計：支援 MDX、自動側邊欄、搜尋整合、多語系、版本控制。零 JS 策略確保文件頁面在任何網路環境下都能快速載入。

行銷與著陸頁

行銷頁面需要極致的載入速度與 SEO 表現。Astro 預設輸出純 HTML，僅在表單或互動展示區域加入 JS 島嶼。搭配 Tailwind 和動畫庫，可建構視覺豐富且效能頂尖的著陸頁。

部落格與媒體網站

Content Collections 提供結構化的內容管理，配合 RSS 整合、Sitemap 生成、Open Graph 圖片生成等功能，建構完整的出版工作流。多框架支援讓團隊可在文章中嵌入互動式圖表或示範元件。
