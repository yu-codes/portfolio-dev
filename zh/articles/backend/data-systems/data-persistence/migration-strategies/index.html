Schema Migration

Schema Migration жҳҜжҢҮиіҮж–ҷеә«зөҗж§Ӣзҡ„зүҲжң¬еҢ–з®ЎзҗҶпјҢеҢ…еҗ«пјҡ

  • зүҲжң¬жҺ§еҲ¶пјҡжҜҸж¬Ў schema и®ҠжӣҙйғҪжңүе°ҚжҮүзҡ„йҒ·з§»и…іжң¬пјҲup/downпјү
  • еҶӘзӯүжҖ§пјҡйҒ·з§»и…іжң¬еҸҜе®үе…ЁйҮҚиӨҮеҹ·иЎҢиҖҢдёҚз”ўз”ҹеүҜдҪңз”Ё
  • дҫқиіҙз®ЎзҗҶпјҡйҒ·з§»д№Ӣй–“зҡ„е…ҲеҫҢй ҶеәҸеҝ…й ҲжҳҺзўәе®ҡзҫ©
  • еӣһж»ҫиғҪеҠӣпјҡжҜҸеҖӢйҒ·з§»йғҪжҮүжңүе°ҚжҮүзҡ„еҸҚеҗ‘ж“ҚдҪң

еёёиҰӢйҒ·з§»е·Ҙе…·

  • FlywayпјҡJava з”ҹж…Ӣзі»дё»жөҒпјҢж”ҜжҸҙ SQL иҲҮ Java йҒ·з§»
  • Liquibaseпјҡж”ҜжҸҙ XML/YAML/JSON жҸҸиҝ° schema и®Ҡжӣҙ
  • AlembicпјҡPython SQLAlchemy з”ҹж…Ӣзі»зҡ„йҒ·з§»е·Ҙе…·
  • golang-migrateпјҡGo з”ҹж…Ӣзі»зҡ„иј•йҮҸйҒ·з§»е·Ҙе…·
  • Prisma MigrateпјҡTypeScript/Node.js з”ҹж…Ӣзі»зҡ„зҸҫд»ЈйҒ·з§»е·Ҙе…·

йҒ·з§»еҲҶйЎһ

  1. зөҗж§ӢйҒ·з§»пјҲDDL Migrationпјүпјҡе»әиЎЁгҖҒж”№ж¬„дҪҚгҖҒеҠ зҙўеј•
  2. иіҮж–ҷйҒ·з§»пјҲData MigrationпјүпјҡиҪүжҸӣгҖҒеЎ«е……гҖҒжё…зҗҶиіҮж–ҷ
  3. е№іеҸ°йҒ·з§»пјҲPlatform MigrationпјүпјҡеҫһдёҖеҖӢиіҮж–ҷеә«зі»зөұйҒ·з§»еҲ°еҸҰдёҖеҖӢ

Expand-Contract Pattern

йҖҷжҳҜеҜҰзҸҫйӣ¶еҒңж©ҹйҒ·з§»зҡ„ж ёеҝғжЁЎејҸпјҡ

  1. Expand йҡҺж®өпјҡж–°еўһж¬„дҪҚжҲ–иЎЁпјҢдёҚз§»йҷӨд»»дҪ•зҸҫжңүзөҗж§Ӣ
  2. йӣҷеҜ«йҡҺж®өпјҡжҮүз”ЁзЁӢејҸеҗҢжҷӮеҜ«е…Ҙж–°иҲҠж¬„дҪҚ
  3. еӣһеЎ«йҡҺж®өпјҡе°Үжӯ·еҸІиіҮж–ҷйҒ·з§»еҲ°ж–°ж¬„дҪҚ
  4. еҲҮжҸӣйҡҺж®өпјҡжҮүз”ЁзЁӢејҸж”№зӮәи®ҖеҸ–ж–°ж¬„дҪҚ
  5. Contract йҡҺж®өпјҡзўәиӘҚз„Ўе•ҸйЎҢеҫҢз§»йҷӨиҲҠж¬„дҪҚ

е ҙжҷҜдёҖпјҡеӨ§еһӢиЎЁзөҗж§Ӣи®Ҡжӣҙ

йңҖиҰҒзӮәдёҖејөж“Ғжңүж•ёе„„зӯҶиЁҳйҢ„зҡ„иЎЁж–°еўһж¬„дҪҚдёҰиЁӯе®ҡй җиЁӯеҖјпјҡ

  1. ALTER TABLE ADD COLUMN ... DEFAULT NULLпјҲзһ¬й–“е®ҢжҲҗпјү
  2. иғҢжҷҜзЁӢејҸжү№ж¬ЎеӣһеЎ«иіҮж–ҷпјҲеҲҶжү№иҷ•зҗҶйҒҝе…ҚйҺ–иЎЁпјү
  3. жҮүз”ЁзЁӢејҸй–Ӣе§ӢдҪҝз”Ёж–°ж¬„дҪҚ
  4. зўәиӘҚжүҖжңүиЁҳйҢ„е·ІеЎ«е……еҫҢеҠ е…Ҙ NOT NULL зҙ„жқҹ

е ҙжҷҜдәҢпјҡи·ЁиіҮж–ҷеә«е№іеҸ°йҒ·з§»

еҫһ MySQL йҒ·з§»иҮі PostgreSQLпјҡ

  1. дҪҝз”Ё pgloader жҲ– AWS DMS йҖІиЎҢеҲқе§ӢиіҮж–ҷеҗҢжӯҘ
  2. иЁӯе®ҡжҢҒзәҢиӨҮиЈҪпјҲCDCпјүдҝқжҢҒе…©з«ҜеҗҢжӯҘ
  3. жҮүз”ЁзЁӢејҸеҜҰдҪңйӣҷи®Җзӯ–з•Ҙй©—иӯүдёҖиҮҙжҖ§
  4. зўәиӘҚз„Ўе·®з•°еҫҢеҲҮжҸӣеҜ«е…ҘиҮіж–°иіҮж–ҷеә«
  5. з¶ӯжҢҒиҲҠеә«зӮәе”Ҝи®ҖдёҖж®өжҷӮй–“дҪңзӮәеӣһйҖҖж–№жЎҲ

е ҙжҷҜдёүпјҡеҫ®жңҚеӢҷиіҮж–ҷеә«жӢҶеҲҶ

е°Үе–®й«”иіҮж–ҷеә«жӢҶеҲҶзӮәеҗ„жңҚеӢҷзҚЁз«ӢиіҮж–ҷеә«пјҡ

  1. иӯҳеҲҘжңҚеӢҷйӮҠз•ҢиҲҮиіҮж–ҷж“Ғжңүж¬Ҡ
  2. е»әз«Ӣж–°иіҮж–ҷеә«дёҰиЁӯе®ҡйӣҷеҜ«
  3. жјёйҖІејҸе°Үи®ҖеҸ–жөҒйҮҸе°Һеҗ‘ж–°еә«
  4. е»әз«Ӣи·ЁжңҚеӢҷиіҮж–ҷеҗҢжӯҘж©ҹеҲ¶пјҲдәӢд»¶й©…еӢ•пјү
  5. з§»йҷӨиҲҠеә«дёӯе·ІжӢҶеҮәзҡ„иЎЁ

еёёиҰӢйҷ·йҳұ

  1. жңӘиҖғж…®йҺ–иЎЁеҪұйҹҝпјҡеӨ§иЎЁ DDL ж“ҚдҪңеҸҜиғҪй•·жҷӮй–“йҺ–иЎЁ
