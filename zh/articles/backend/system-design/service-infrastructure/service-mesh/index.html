Service Mesh жҳҜдёҖеҖӢе°Ҳй–Җиҷ•зҗҶжңҚеӢҷй–“йҖҡиЁҠзҡ„еҹәзӨҺиЁӯж–ҪеұӨгҖӮйҖҡйҒҺе°Үз¶Іи·ҜйӮҸијҜеҫһжҮүз”ЁзЁӢејҸзўјдёӯжҠҪйӣўпјҢService Mesh зӮәеҫ®жңҚеӢҷжһ¶ж§ӢжҸҗдҫӣзөұдёҖзҡ„жөҒйҮҸз®ЎзҗҶгҖҒе®үе…ЁжҖ§е’ҢеҸҜи§Җжё¬жҖ§пјҢиҖҢз„ЎйңҖдҝ®ж”№жҘӯеӢҷзЁӢејҸзўјгҖӮ

ж ёеҝғжҰӮеҝө

д»ҖйәјжҳҜ Service Mesh

Service Mesh жҳҜдёҖеҖӢеҸҜй…ҚзҪ®зҡ„еҹәзӨҺиЁӯж–ҪеұӨпјҢз”Ёж–јиҷ•зҗҶеҫ®жңҚеӢҷд№Ӣй–“зҡ„йҖҡиЁҠгҖӮе®ғйҖҡеёёд»Ҙ Sidecar Proxy зҡ„еҪўејҸйғЁзҪІеңЁжҜҸеҖӢжңҚеӢҷеҜҰдҫӢж—ҒйӮҠпјҢж””жҲӘдёҰз®ЎзҗҶжүҖжңүйҖІеҮәи©ІжңҚеӢҷзҡ„з¶Іи·ҜжөҒйҮҸгҖӮ

ж ёеҝғиғҪеҠӣ

  • жөҒйҮҸз®ЎзҗҶпјҲTraffic Managementпјүпјҡи·Ҝз”ұиҰҸеүҮгҖҒйҮ‘зөІйӣҖзҷјеёғгҖҒж•…йҡңжіЁе…ҘгҖҒйҮҚи©Ұзӯ–з•Ҙ
  • е®үе…ЁжҖ§пјҲSecurityпјүпјҡmTLS иҮӘеӢ•еҠ еҜҶгҖҒжңҚеӢҷиә«д»ҪиӘҚиӯүгҖҒеӯҳеҸ–жҺ§еҲ¶ж”ҝзӯ–
  • еҸҜи§Җжё¬жҖ§пјҲObservabilityпјүпјҡеҲҶж•ЈејҸиҝҪи№ӨгҖҒжҢҮжЁҷж”¶йӣҶгҖҒеӯҳеҸ–ж—ҘиӘҢ
  • еҪҲжҖ§пјҲResilienceпјүпјҡзҶ”ж–·гҖҒйҖҫжҷӮгҖҒйҮҚи©ҰгҖҒйҷҗжөҒ

жһ¶ж§ӢжЁЎејҸ

Sidecar жЁЎејҸпјҡжҜҸеҖӢжңҚеӢҷеҜҰдҫӢж—ҒйғҪйғЁзҪІдёҖеҖӢд»ЈзҗҶпјҲеҰӮ EnvoyпјүпјҢжүҖжңүжөҒйҮҸйҖҡйҒҺд»ЈзҗҶиҪүзҷјгҖӮйҖҷжҳҜзӣ®еүҚжңҖдё»жөҒзҡ„еҜҰзҸҫж–№ејҸгҖӮ

Proxyless жЁЎејҸпјҡе°Ү Mesh йӮҸијҜд»ҘеҮҪејҸеә«еҪўејҸж•ҙеҗҲеҲ°жҮүз”ЁдёӯпјҲеҰӮ gRPC зҡ„ xDS ж”ҜжҸҙпјүпјҢжёӣе°‘е»¶йҒІдҪҶеўһеҠ иӘһиЁҖз¶Ғе®ҡгҖӮ

Ambient MeshпјҡIstio зҡ„ж–°жЁЎејҸпјҢдҪҝз”Ёе…ұдә«зҡ„ ztunnel зҜҖй»һд»ЈзҗҶеҸ–д»Ј per-pod sidecarпјҢйҷҚдҪҺиіҮжәҗй–ӢйҠ·гҖӮ

Service Mesh vs API Gateway

йқўеҗ‘Service MeshAPI Gateway
жөҒйҮҸж–№еҗ‘жқұиҘҝеҗ‘пјҲжңҚеӢҷй–“пјүеҚ—еҢ—еҗ‘пјҲеӨ–йғЁеҲ°е…§йғЁпјү
йғЁзҪІдҪҚзҪ®жҜҸеҖӢжңҚеӢҷж—ҒеҸўйӣҶйӮҠз•Ң
дё»иҰҒй—ңжіЁжңҚеӢҷй–“йҖҡиЁҠжІ»зҗҶеӨ–йғЁ API з®ЎзҗҶ
иӘҚиӯүж–№ејҸmTLSOAuth2/JWT

жҳҜеҗҰйңҖиҰҒ Service Meshпјҹ

  • йңҖиҰҒпјҡеӨ§йҮҸеҫ®жңҚеӢҷпјҲ50+пјүгҖҒеӨҡиӘһиЁҖжҠҖиЎ“жЈ§гҖҒеҡҙж јзҡ„е®үе…ЁеҗҲиҰҸйңҖжұӮ
  • еҸҜиғҪдёҚйңҖиҰҒпјҡжңҚеӢҷж•ёйҮҸе°‘пјҲ<10пјүгҖҒе–®дёҖиӘһиЁҖжЈ§гҖҒеңҳйҡҠиҰҸжЁЎе°Ҹ

жҮүз”Ёе ҙжҷҜ

йӣ¶дҝЎд»»е®үе…Ёжһ¶ж§Ӣ

Service Mesh зҡ„ mTLS еҠҹиғҪиҮӘеӢ•еҠ еҜҶжүҖжңүжңҚеӢҷй–“йҖҡиЁҠпјҢз„ЎйңҖдҝ®ж”№жҮүз”ЁзЁӢејҸзўјгҖӮй…ҚеҗҲжңҚеӢҷиә«д»Ҫе’Ң RBAC ж”ҝзӯ–пјҢеҜҰзҸҫзҙ°зІ’еәҰзҡ„йӣ¶дҝЎд»»еӯҳеҸ–жҺ§еҲ¶пјҢж»ҝи¶і SOC2 е’Ң PCI DSS еҗҲиҰҸиҰҒжұӮгҖӮ

йҮ‘зөІйӣҖзҷјеёғиҲҮжөҒйҮҸеҲҮеҲҶ

йҖҡйҒҺ VirtualService е’Ң DestinationRule еҜҰзҸҫзІҫзҙ°зҡ„жөҒйҮҸжҺ§еҲ¶гҖӮдҫӢеҰӮе°Ү 5% зҡ„жөҒйҮҸе°Һеҗ‘ж–°зүҲжң¬пјҢи§ҖеҜҹйҢҜиӘӨзҺҮе’Ңе»¶йҒІпјҢйҖҗжӯҘеўһеҠ жҜ”дҫӢгҖӮеҰӮжһңеҮәзҸҫе•ҸйЎҢпјҢз«ӢеҚіеӣһж»ҫиҖҢз„ЎйңҖйҮҚж–°йғЁзҪІгҖӮ

еӨҡеҸўйӣҶжңҚеӢҷжІ»зҗҶ

и·ЁеӨҡеҖӢ Kubernetes еҸўйӣҶзҡ„жңҚеӢҷйңҖиҰҒзөұдёҖзҡ„жөҒйҮҸз®ЎзҗҶе’Ңе®үе…Ёзӯ–з•ҘгҖӮIstio Multi-Cluster жҲ– Consul зҡ„ WAN Federation жҸҗдҫӣи·ЁеҸўйӣҶзҡ„жңҚеӢҷзҷјзҸҫе’ҢжөҒйҮҸи·Ҝз”ұгҖӮ

ж•…йҡңжіЁе…Ҙжё¬и©Ұ

еңЁз”ҹз”ўз’°еўғдёӯжіЁе…Ҙе»¶йҒІжҲ–йҢҜиӘӨпјҢжё¬и©Ұзі»зөұзҡ„йҹҢжҖ§гҖӮдҫӢеҰӮе°Қзү№е®ҡжңҚеӢҷзҡ„ 10% и«ӢжұӮжіЁе…Ҙ 5 з§’е»¶йҒІпјҢй©—иӯүдёҠжёёжңҚеӢҷзҡ„йҖҫжҷӮе’ҢйҮҚи©Ұж©ҹеҲ¶жҳҜеҗҰжӯЈеёёйҒӢдҪңгҖӮ

еҸҜи§Җжё¬жҖ§зөұдёҖ

з„ЎйңҖдҝ®ж”№зЁӢејҸзўјеҚіеҸҜзҚІеҫ—е®Ңж•ҙзҡ„еҲҶж•ЈејҸиҝҪи№ӨпјҲJaeger/ZipkinпјүгҖҒжҢҮжЁҷпјҲPrometheusпјүе’ҢеӯҳеҸ–ж—ҘиӘҢгҖӮжүҖжңүжңҚеӢҷзҡ„ RED жҢҮжЁҷпјҲRateгҖҒErrorгҖҒDurationпјүиҮӘеӢ•ж”¶йӣҶдёҰеҸҜиҰ–еҢ–гҖӮ
