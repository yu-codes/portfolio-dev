後端工程

涵蓋從程式語言基礎到正式環境系統的完整後端工程知識，包括 API、資料庫、系統設計與維運。

後端工程是現代軟體系統的核心骨幹。作為一名後端工程師，你需要負責建構伺服器端邏輯、設計可擴展架構、管理資料庫，並確保系統能在正式環境中穩定可靠地運行。

本章節提供一條具結構性的學習路徑，帶你從程式設計基礎一路進展到可操作大型分散式系統的工程能力。

為什麼是後端工程？

你每天使用的所有應用程式——從社群媒體到銀行系統——都依賴後端系統運作。後端工程師需要解決軟體工程中最具挑戰性的問題之一：

  • 如何處理每秒數百萬級的請求？
  • 如何在分散式系統中維持資料一致性？
  • 如何設計同時具備彈性與易用性的 API？
  • 如何讓系統全年無休穩定運作並將停機時間降到最低？

子領域結構

每個領域都會再細分為 2 至 5 個子領域，以涵蓋更具體的技能範疇。

BE1：語言與基礎能力

  • 核心語言特性（Python、Go、Java、Node.js）
  • 物件導向與函數式程式設計
  • 記憶體管理與並行處理
  • 程式碼品質與最佳實踐

BE2：資料結構與演算法

  • 陣列、鏈結串列、樹、圖
  • 排序與搜尋演算法
  • 時間與空間複雜度分析
  • 問題拆解與解題模式

BE3：API 與通訊協定

  • RESTful API 設計原則
  • GraphQL Schema 設計
  • gRPC 與 Protocol Buffers
  • WebSocket 與即時通訊

BE4：資料庫與儲存

  • 關聯式資料庫設計與 SQL
  • NoSQL 資料庫（MongoDB、Redis、Cassandra）
  • 索引與查詢優化
  • 資料建模與正規化

BE5：系統設計與架構

  • 微服務與單體架構比較
  • 設計模式（CQRS、Event Sourcing）
  • 訊息佇列與事件驅動系統
  • CAP 定理與分散式系統

BE6：可擴展性與效能

  • 水平與垂直擴展
  • 快取策略（Redis、Memcached）
  • 資料庫分片與複寫
  • 效能分析與優化

BE7：資訊安全

  • 身分驗證（OAuth、JWT、SAML）
  • 授權與存取控制
  • 靜態與傳輸中的資料加密
  • OWASP 安全實踐

BE8：測試與品質保證

  • 單元測試框架與實踐
  • 整合測試與端對端測試
  • 測試驅動開發（TDD）
  • 程式碼覆蓋率與品質指標

BE9：部署與 CI/CD

  • Docker 容器化
  • Kubernetes 編排
  • CI/CD Pipeline 設計
  • 藍綠部署與 Canary Deployment

BE10：可觀測性與維運

  • 日誌策略與集中化管理
  • 指標監控（Prometheus、Grafana）
  • 分散式追蹤（Distributed Tracing）
  • Incident Response 與 On-call 實務
