過擬合與正則化

過擬合是機器學習中最常見也最棘手的問題之一——模型在訓練集上表現優異，但在未見過的新資料上效能急劇下降。正則化技術則是對抗過擬合的主要武器，透過限制模型的複雜度來提升泛化能力。本文將深入探討過擬合的成因、診斷方法以及各種正則化策略。

核心概念

過擬合的本質

過擬合（Overfitting）發生在模型學到了訓練資料中的噪音和個別樣本的特異性，而非底層的真實規律。從統計學習理論的角度，這是模型複雜度超過了資料所能支撐的程度。

過擬合的信號：

  • 訓練損失持續下降，但驗證損失在某個時間點後開始上升
  • 訓練集精度 99%，測試集精度 80%（巨大的泛化差距）
  • 模型權重值很大（線性模型）或決策邊界極度扭曲（非線性模型）
  • 微小的輸入變化導致預測結果大幅波動

過擬合的成因：

  • 模型過於複雜：參數數量遠超訓練樣本數
  • 訓練資料不足：樣本不足以覆蓋問題的複雜度
  • 訓練時間過長：在有限資料上過度優化
  • 噪音資料：資料中包含標註錯誤或不可預測的隨機性
  • 特徵過多：包含不相關或冗餘的特徵

欠擬合的對照

欠擬合（Underfitting）是另一個極端——模型太簡單，無法捕捉資料中的有用模式：

  • 訓練集和測試集的表現都很差
  • 模型過度簡化了問題的結構
  • 常見原因：特徵不足、模型容量太小、正則化過強

正則化技術分類

參數正則化（Parameter Regularization）：

  • L2 正則化（Ridge / Weight Decay）： 損失函數加上 λΣwᵢ²。效果是將所有權重均勻地縮小（shrinkage），但不會壓縮到零。幾何上等價於將解限制在一個球體內。

  • L1 正則化（Lasso）： 損失函數加上 λΣ|wᵢ|。效果是將部分權重壓縮為精確的零，自動實現特徵選擇。幾何上等價於將解限制在一個菱形內。

  • Elastic Net： 結合 L1 和 L2：λ₁Σ|wᵢ| + λ₂Σwᵢ²。既能選擇特徵又能處理相關特徵群組。

結構正則化（Structural Regularization）：

  • Dropout：訓練時隨機關閉一定比例的神經元（典型值 0.1-0.5）。迫使網路學習分散的、穩健的表示。測試時使用所有神經元但縮放輸出。可以理解為隱式的模型集成。

  • Batch Normalization：標準化每層的輸入分佈，減少內部協變量偏移。雖然主要目的是加速訓練，但也有輕微的正則化效果（因為 batch 統計量的隨機性）。

  • Layer Normalization：在每個樣本的特徵維度上做正規化，不依賴 batch 大小，適合序列模型。

資料正則化（Data Regularization）：

  • 資料增強（Data Augmentation）：通過對訓練資料施加保留語義的變換（旋轉、裁剪、翻轉、色彩抖動）來增加有效樣本數
  • Mixup / CutMix：將兩個樣本及其標籤進行線性混合或區域替換，創造虛擬訓練樣本
  • Label Smoothing：將 hard label（0/1）替換為 soft label（0.1/0.9），防止模型過度自信

訓練策略正則化：

  • Early Stopping：在驗證損失不再下降時停止訓練。最簡單有效的正則化方法之一
  • 學習率排程（LR Schedule）：Warmup + Cosine Decay，避免過快收斂到局部最優
  • 梯度裁剪（Gradient Clipping）：限制梯度的最大值，防止梯度爆炸

架構總覽

過擬合診斷與處理的完整流程：

  1. 監控學習曲線：

    • 繪製訓練/驗證損失隨 epoch 的變化
    • 識別過擬合開始的時間點
    • 比較不同模型複雜度的學習曲線

  2. 診斷分析：

    • 計算泛化差距（generalization gap）
    • 分析模型是高偏差還是高方差
    • 檢查資料品質和標籤一致性

  3. 選擇策略：

    • 高方差 → 增加正則化或減少模型複雜度
    • 高偏差 → 增加模型複雜度或減少正則化
    • 資料不足 → 資料增強或遷移學習

  4. 正則化強度調整：

    • 使用交叉驗證搜尋最佳正則化超參數
    • 觀察正則化路徑（如 Lasso 的正則化路徑圖）
    • 平衡訓練集表現與泛化差距

L1 vs L2 正則化的選擇：

  • L1：當你預期只有少數特徵真正重要時（稀疏解）
  • L2：當你預期所有特徵都有一定貢獻時（均勻縮小）
  • 不確定時使用 Elastic Net

Dropout vs Batch Normalization：

  • 兩者通常不同時使用（BN 的統計量會受 Dropout 影響）
  • 現代做法：在 ResNet 中主要用 BN，在 Transformer 中用 Dropout + LayerNorm
  • Dropout 在全連接層效果顯著，在卷積層效果有限

Weight Decay vs L2 正則化：

  • 在標準 SGD 中兩者等價
  • 在 Adam 中不等價：AdamW 解耦了權重衰減與自適應學習率
  • 實務中大型模型訓練應使用 AdamW 而非 Adam + L2

應用場景

場景一：深度學習影像分類

使用 ResNet-50 在 5000 張醫療影像上訓練。正則化組合：

  • 資料增強（旋轉±15°、水平翻轉、色彩抖動）
  • 預訓練權重 + 低學習率微調（遷移學習本身就是正則化）
  • Cosine LR Schedule + Early Stopping（patience=10）
  • Label Smoothing ε=0.1

場景二：自然語言處理微調

BERT 在下游小資料集上微調容易過擬合。策略：

  • 低學習率（2e-5 ~ 5e-5）
  • 線性 warmup + 線性衰減
  • Dropout = 0.1（BERT 內建）
  • 訓練 3-5 個 epoch 即可（不宜過多）
  • R-Drop：對同一輸入兩次前向傳播的輸出做一致性正則化

場景三：表格資料的 XGBoost

XGBoost 內建多種正則化：

  • max_depth：限制樹的深度（通常 4-8）
  • min_child_weight：葉節點最小樣本數
  • subsample + colsample_bytree：隨機子採樣（類似 Dropout）
  • reg_alpha（L1）+ reg_lambda（L2）：葉節點權重的正則化
  • early_stopping_rounds：驗證集表現不再提升時停止

場景四：時間序列預測

LSTM/Transformer 在有限歷史資料上容易過擬合：

  • 使用較短的 lookback window
  • Recurrent Dropout（在時間步之間保持相同的 dropout mask）
  • 時間序列特定的資料增強（加噪、時間扭曲）
  • 使用更簡單的模型（如 LightGBM with lag features）作為 baseline
