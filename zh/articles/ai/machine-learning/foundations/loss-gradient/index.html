損失函數與梯度下降

機器學習的訓練過程可以歸結為一個優化問題：定義一個損失函數來量化模型預測與真實標籤之間的差距，然後透過梯度下降等優化算法來最小化這個損失。本文將深入探討損失函數的設計原理以及各種梯度下降變體的運作機制。

核心概念

損失函數的本質

損失函數（Loss Function）是一個將模型預測映射到非負實數的函數，值越小代表模型表現越好。它是連接「模型做了什麼」與「模型應該做什麼」的橋梁。

回歸任務常用損失函數：

  • 均方誤差（MSE）：L = (1/n)Σ(yᵢ - ŷᵢ)²，對大誤差懲罰更重，對離群值敏感
  • 平均絕對誤差（MAE）：L = (1/n)Σ|yᵢ - ŷᵢ|，對離群值更穩健，但在零點不可微
  • Huber Loss：結合 MSE 和 MAE 的優點，在小誤差時使用 MSE，大誤差時使用 MAE

分類任務常用損失函數：

  • 二元交叉熵（Binary Cross-Entropy）：L = -[y·log(ŷ) + (1-y)·log(1-ŷ)]，用於二分類問題
  • 類別交叉熵（Categorical Cross-Entropy）：多分類的推廣，配合 softmax 輸出使用
  • Hinge Loss：max(0, 1 - y·ŷ)，SVM 的核心損失函數，鼓勵分類邊界最大化

梯度的意義

梯度（Gradient）是多變數函數在某點的方向導數中變化最快的方向。對損失函數 L 關於參數 θ 求梯度 ∇L(θ)，得到的向量指向損失增加最快的方向。因此，沿著梯度的反方向更新參數，就能使損失逐步減小。

梯度下降的更新規則：θ_new = θ_old - η · ∇L(θ_old)

其中 η 為學習率（learning rate），控制每次更新的步幅。

反向傳播

反向傳播（Backpropagation）是高效計算深度神經網路梯度的算法。它利用鏈式法則（Chain Rule），從輸出層開始逐層向輸入層傳遞梯度，避免了重複計算。

計算圖（Computational Graph）是理解反向傳播的關鍵：

  • 前向傳播：按拓撲序計算每個節點的值
  • 反向傳播：按逆拓撲序計算每個節點對損失的梯度

架構總覽

梯度下降的優化流程可以分為以下層次：

  1. 損失計算：前向傳播得到預測值，計算損失函數值
  2. 梯度計算：反向傳播計算所有參數的梯度
  3. 參數更新：根據優化策略更新參數值
  4. 收斂判斷：檢查是否滿足停止條件（損失收斂、達到最大迭代次數等）

梯度下降的三種變體：

  • 批次梯度下降（Batch GD）：使用全部訓練資料計算梯度，更新穩定但計算量大
  • 隨機梯度下降（SGD）：每次只用一個樣本計算梯度，更新頻繁但噪音大
  • 小批次梯度下降（Mini-batch GD）：折衷方案，使用一小批資料計算梯度，是實務中最常用的方法

進階優化器：

  • Momentum：累積過去梯度的動量，加速收斂並減少震盪
  • AdaGrad：自適應學習率，對稀疏特徵使用較大學習率
  • RMSProp：改進 AdaGrad，使用指數移動平均避免學習率過快衰減
  • Adam：結合 Momentum 和 RMSProp，是目前最流行的優化器

關鍵比較

優化器學習率自適應動量機制適用場景記憶體需求
SGD無無凸優化問題低
SGD + Momentum無一階動量高曲率平面低
AdaGrad累積平方梯度無稀疏特徵中
RMSProp指數衰減平均無非穩態問題中
Adam指數衰減平均一階+二階動量通用場景高
AdamW解耦權重衰減一階+二階動量大型模型微調高

MSE vs Cross-Entropy 在分類問題中的比較：

  • MSE 在 sigmoid 輸出接近 0 或 1 時梯度極小，導致學習緩慢
  • Cross-Entropy 在預測錯誤時產生大梯度，學習信號更強
  • 因此分類問題應優先使用 Cross-Entropy

學習率的影響：

  • 太大：損失震盪甚至發散
  • 太小：收斂極慢，容易卡在局部最小值
  • 最佳實踐：使用學習率排程（warmup + cosine decay）

應用場景

場景一：影像分類中的損失設計

在 ImageNet 分類任務中，標準做法是使用 softmax + cross-entropy loss。但在面對類別不平衡時，可以採用 Focal Loss（γ 參數加大對困難樣本的關注），或使用 Label Smoothing 來防止模型過度自信。

場景二：目標檢測的多任務損失

YOLO 等目標檢測模型需要同時優化：

  • 定位損失（IoU Loss / GIoU Loss）：物體框的位置準確度
  • 置信度損失（Binary Cross-Entropy）：是否包含物體
  • 分類損失（Cross-Entropy）：物體類別

多任務損失的權重平衡是一個重要的超參數調整課題。

場景三：大型語言模型的預訓練

GPT 系列模型使用自迴歸語言建模損失（next token prediction），本質上是序列中每個位置的 cross-entropy loss 的平均。配合 AdamW 優化器和 cosine 學習率排程，在數兆 token 的資料上進行訓練。

場景四：對比學習中的 InfoNCE Loss

在自監督學習中，InfoNCE Loss 拉近正樣本對、推遠負樣本對的距離。這種損失函數設計不需要顯式標籤，是 SimCLR、CLIP 等方法的核心。
