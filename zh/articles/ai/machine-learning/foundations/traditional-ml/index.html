傳統機器學習演算法

在深度學習興起之前，傳統機器學習演算法已經在眾多實際問題中取得了巨大的成功。即使在今天，面對結構化資料、小樣本場景或需要可解釋性的應用，傳統方法仍然是首選。本文將系統性地介紹監督式與非監督式學習中的經典演算法。

核心概念

監督式學習

監督式學習從標註資料中學習輸入到輸出的映射關係。根據輸出類型，可分為分類（離散標籤）和回歸（連續數值）。

線性模型：

  • 線性回歸（Linear Regression）：假設輸出是輸入特徵的線性組合，透過最小化 MSE 求解。具有封閉解（正規方程）或可用梯度下降求解
  • 邏輯回歸（Logistic Regression）：在線性模型後加上 sigmoid 函數，輸出機率值。儘管名稱帶有「回歸」，實際上是分類方法
  • 支持向量機（SVM）：找到最大化分類邊界的超平面。核心思想是最大間隔原則，透過核函數可處理非線性可分問題

樹模型：

  • 決策樹（Decision Tree）：透過遞迴分裂特徵空間來建立分類或回歸規則。分裂準則包括資訊增益（ID3）、增益比（C4.5）和基尼不純度（CART）
  • 隨機森林（Random Forest）：集成多棵決策樹，透過 Bagging 和特徵隨機子集來降低過擬合
  • 梯度提升樹（Gradient Boosting）：序列式地訓練弱學習器，每棵新樹擬合前一輪的殘差。XGBoost、LightGBM、CatBoost 是其高效實現

最近鄰方法：

  • K-最近鄰（KNN）：不建立模型，直接根據最近的 K 個鄰居投票預測。簡單但計算量隨資料量線性增長

非監督式學習

非監督式學習從未標註資料中發現結構和模式。

聚類（Clustering）：

  • K-Means：將資料劃分為 K 個簇，最小化簇內距離平方和。需預設 K 值，對初始化敏感
  • DBSCAN：基於密度的聚類，能發現任意形狀的簇，自動識別噪音點
  • 階層式聚類（Hierarchical Clustering）：自底向上合併或自頂向下分裂，產生樹狀圖（dendrogram）

降維（Dimensionality Reduction）：

  • 主成分分析（PCA）：線性降維，投影到最大方差方向
  • t-SNE：非線性降維，保留局部鄰域結構，適合高維資料視覺化
  • UMAP：比 t-SNE 更快且保留更多全局結構

集成學習

集成學習結合多個基學習器的預測來提升整體表現：

  • Bagging：並行訓練多個模型，透過投票/平均聚合（隨機森林）
  • Boosting：序列式訓練，後續模型修正前面模型的錯誤（AdaBoost、GBM）
  • Stacking：用一個元模型學習如何組合基模型的預測

架構總覽

傳統機器學習的工作流程：

  1. 資料收集與清洗：處理缺失值、異常值、格式不一致
  2. 特徵工程：建立有意義的特徵，包括數值標準化、類別編碼、交互特徵
  3. 模型選擇：根據問題特性選擇候選演算法
  4. 交叉驗證：K-fold CV 評估模型泛化能力
  5. 超參數調整：網格搜尋、隨機搜尋或貝氏優化
  6. 模型評估：在測試集上計算最終指標
  7. 部署與監控：上線服務並監控效能衰退

偏差-方差權衡是理解演算法選擇的核心框架：

  • 高偏差（欠擬合）：模型太簡單，無法捕捉資料模式
  • 高方差（過擬合）：模型太複雜，對訓練資料過度敏感
  • 理想模型在兩者之間取得平衡

關鍵比較

演算法可解釋性訓練速度預測速度處理非線性特徵工程需求
線性回歸高快快差高
邏輯回歸高快快差高
SVM中慢(大資料)快好(核函數)中
決策樹高快快好低
隨機森林中中中好低
XGBoost中中快好低
KNN低無慢好高

隨機森林 vs XGBoost：

  • 隨機森林：平行訓練、不易過擬合、超參數少
  • XGBoost：通常精度更高、提供正則化選項、支援缺失值、需要更仔細的超參數調整

生成式模型 vs 判別式模型：

  • 生成式（Naive Bayes、GMM）：學習聯合分佈 P(X,Y)，可生成新樣本
  • 判別式（SVM、LR）：直接學習條件分佈 P(Y|X)，通常分類精度更高

應用場景

場景一：信用卡欺詐偵測

使用 XGBoost 或 LightGBM 處理高度不平衡的交易資料。搭配 SMOTE 過採樣、調整類別權重、使用 AUC-PR 作為評估指標。樹模型能處理混合類型特徵且提供特徵重要性排名。

場景二：客戶分群

使用 K-Means 或 DBSCAN 對客戶行為進行聚類分析。結合 RFM（Recency、Frequency、Monetary）特徵，識別高價值客戶群體，制定差異化行銷策略。

場景三：房價預測

使用隨機森林或梯度提升回歸處理結構化的房屋特徵資料（面積、房間數、地段等）。相比深度學習，樹模型在中小型表格資料上通常表現更好且更快。

場景四：文件分類

使用 TF-IDF 向量化配合邏輯回歸或 SVM 進行文件分類。在標註資料有限的場景下，這種組合的效果常常優於需要大量資料的深度學習方法。
