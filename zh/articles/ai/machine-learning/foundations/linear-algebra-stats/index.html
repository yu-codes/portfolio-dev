線性代數與統計基礎

機器學習的核心建立在數學基礎之上，其中線性代數與統計學扮演著不可或缺的角色。從資料表示到模型訓練，幾乎每個環節都涉及向量、矩陣運算以及機率分佈的概念。本文將系統性地介紹這些基礎知識，幫助讀者建立紮實的數學直覺。

核心概念

向量與矩陣

在機器學習中，資料通常以向量和矩陣的形式表示。一個資料點可以視為 n 維空間中的一個向量，而整個資料集則構成一個矩陣，其中每一行代表一個樣本，每一列代表一個特徵。

向量運算包括：

  • 內積（Dot Product）：衡量兩個向量的相似度，廣泛用於相似性計算與投影
  • 外積（Cross Product）：產生新的向量空間，用於幾何變換
  • 範數（Norm）：衡量向量的大小，L1 範數與 L2 範數分別對應正則化技術

矩陣運算包括：

  • 矩陣乘法：線性變換的核心，神經網路的前向傳播本質上就是連續的矩陣乘法
  • 轉置（Transpose）：行列互換，在梯度計算中頻繁使用
  • 逆矩陣（Inverse）：求解線性方程組，用於最小平方法的封閉解

特徵值與特徵向量

特徵值分解（Eigendecomposition）揭示了矩陣的內在結構。對於一個方陣 A，若存在向量 v 和純量 λ 使得 Av = λv，則 v 為特徵向量，λ 為對應的特徵值。

這個概念在以下場景中至關重要：

  • 主成分分析（PCA）：找到資料變異最大的方向，實現降維
  • 協方差矩陣分析：理解特徵之間的關聯性
  • 奇異值分解（SVD）：推薦系統與自然語言處理中的矩陣分解技術

機率與統計

機率分佈是描述隨機變數行為的數學工具：

  • 常態分佈（Gaussian）：自然界中最常見的分佈，中央極限定理的基礎
  • 伯努利分佈（Bernoulli）：二元分類問題的基礎
  • 多項式分佈（Multinomial）：多類別分類的機率模型

條件機率與貝氏定理： 貝氏定理 P(A|B) = P(B|A)P(A)/P(B) 是貝氏分類器與許多生成式模型的理論基礎。它讓我們能夠根據觀察到的證據更新先驗信念。

統計估計：

  • 最大似然估計（MLE）：找到使觀察資料機率最大的參數
  • 最大後驗估計（MAP）：結合先驗知識的參數估計，等價於加入正則化的 MLE

架構總覽

線性代數與統計在機器學習管線中的角色可從以下層次理解：

  1. 資料表示層：將原始資料轉換為向量/矩陣形式，建立特徵空間
  2. 模型計算層：利用矩陣運算實現模型的前向傳播與預測
  3. 優化求解層：透過梯度（向量微積分）與海森矩陣（二階資訊）進行參數更新
  4. 評估分析層：使用統計檢定與信賴區間評估模型表現

在深度學習框架中，張量（Tensor）是向量和矩陣的高維推廣。一張彩色圖片可表示為三維張量（高度 × 寬度 × 通道），而一批圖片則構成四維張量。

關鍵比較

概念線性代數視角統計視角ML 應用
資料表示向量空間中的點隨機變數的實現值特徵矩陣 X
模型參數線性變換的係數待估計的母體參數權重矩陣 W
損失函數距離度量（範數）負對數似然MSE / Cross-entropy
正則化約束解的範數引入先驗分佈L1(Lasso) / L2(Ridge)
降維投影到低維子空間保留最大變異PCA / SVD
相似度餘弦相似度（內積）相關係數推薦系統 / 檢索

L1 vs L2 正則化的幾何直覺：

  • L1 範數約束形成菱形區域，傾向產生稀疏解（特徵選擇）
  • L2 範數約束形成圓形區域，傾向使權重均勻縮小

頻率學派 vs 貝氏學派：

  • 頻率學派：參數是固定未知量，透過 MLE 估計
  • 貝氏學派：參數本身是隨機變數，透過後驗分佈推斷

應用場景

場景一：推薦系統中的矩陣分解

Netflix Prize 競賽中，SVD 矩陣分解技術大放異彩。將使用者-物品評分矩陣分解為使用者潛在因子矩陣與物品潛在因子矩陣的乘積，能有效處理稀疏資料並發現隱含的偏好模式。

場景二：自然語言處理中的向量表示

Word2Vec 和 GloVe 將詞彙映射到稠密向量空間，使得語義相近的詞在向量空間中距離較近。這種表示方式依賴線性代數中的分佈式假設與共現矩陣分解。

場景三：異常檢測中的統計方法

Mahalanobis 距離結合了協方差矩陣的資訊，比歐氏距離更能準確衡量多維資料中的異常程度。這在工業品質監控和金融欺詐偵測中廣泛應用。

場景四：A/B 測試與統計推斷

在模型部署後，需要透過假設檢定（t-test、卡方檢定）來判斷新模型是否顯著優於舊模型。理解 p 值、信賴區間和統計功效對於做出可靠的業務決策至關重要。
