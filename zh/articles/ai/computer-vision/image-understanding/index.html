影像理解

影像分類、偵測與分割

影像理解（Image Understanding）是電腦視覺的基石，涵蓋了讓機器解讀視覺場景的核心能力。從判斷一張圖片「是什麼」，到定位物體「在哪裡」，再到理解每個像素「屬於什麼」，影像理解的三大經典任務——分類、偵測與分割——構成了幾乎所有視覺應用的基礎。本文將深入探討這些技術的原理、演進與實踐方法。

核心概念

影像理解的本質是在不同粒度上對視覺資訊進行語義解析：

影像級別理解：將整張影像對應到一個或多個語義標籤。這是最基礎的理解層次，對應分類（Classification）任務。單標籤分類判斷影像的主要類別，多標籤分類則識別影像中所有存在的概念。

區域級別理解：識別影像中特定區域的物體，並以邊界框（Bounding Box）或旋轉框（Oriented Box）標示其位置。物體偵測（Object Detection）需要同時解決「是什麼」與「在哪裡」兩個問題，難度顯著高於純分類。

像素級別理解：為影像中的每個像素賦予語義標籤。分割（Segmentation）任務提供了最精細的空間理解，是自動駕駛、醫學影像等場景的關鍵技術。

這三個層次之間存在天然的遞進關係：分類提供全域語義、偵測加入空間定位、分割實現精細邊界。現代模型架構（如 Mask R-CNN）能在單一前向傳播中同時輸出多個層次的預測。

架構總覽

影像理解任務共享相似的網路架構設計原則：

輸入 → Backbone（特徵提取）→ Neck（特徵融合）→ Head（任務預測）→ 後處理

分類架構演進：

  • LeNet → AlexNet → VGG → GoogLeNet → ResNet → EfficientNet → ViT → Swin Transformer

偵測架構分支：

  • 兩階段：R-CNN → Fast R-CNN → Faster R-CNN → Cascade R-CNN
  • 單階段：SSD → YOLO 系列 → RetinaNet → FCOS
  • Transformer-based：DETR → Deformable DETR → DINO

分割架構演進：

  • 語義分割：FCN → U-Net → DeepLab 系列 → SegFormer
  • 實例分割：Mask R-CNN → SOLO → QueryInst
  • 全景分割：Panoptic FPN → MaskFormer → Mask2Former

近年最重要的趨勢是統一架構的出現。SAM（Segment Anything Model）展示了一個模型處理多種分割任務的可能性，而 Florence、X-Decoder 等模型則嘗試統一分類、偵測與分割。

關鍵比較

任務輸出形式評估指標典型難點標註成本

訓練資料方面，分類通常需要數千到數十萬張標註影像，偵測需要精確的邊界框標註，而像素級分割的標註成本可達分類的 10-50 倍。因此，半監督學習、自監督預訓練與資料增強在分割任務中尤為重要。

子領域介紹

分類（Classification）

影像分類是深度學習最早突破的視覺任務。2012 年 AlexNet 在 ImageNet 上的成功標誌了深度學習時代的開始。現代分類系統面臨的挑戰包括：

  • 細粒度分類：區分高度相似的子類別（如鳥類品種）
  • 長尾分佈：處理類別嚴重不平衡的現實場景
  • 域適應：模型從訓練域泛化到新環境
  • 開放集辨識：識別訓練時未見過的新類別

偵測（Detection）

物體偵測需要在定位與分類之間取得平衡。現代偵測器的核心設計決策包括：

  • Anchor-based vs Anchor-free：是否使用預定義的參考框
  • 單階段 vs 兩階段：速度與精度的取捨
  • NMS vs End-to-end：後處理策略的選擇

YOLO 系列以其即時推論能力在工業界廣泛應用，而 DETR 系列則以端到端的優雅設計吸引研究社群。

分割（Segmentation）

分割是影像理解中粒度最細的任務，可進一步細分為：

  • 語義分割：每個像素分配類別標籤，不區分實例
  • 實例分割：區分同類別的不同物體實例
  • 全景分割：統一語義分割與實例分割
  • 互動式分割：根據使用者提示（點擊、框選）進行分割

SAM 的出現讓零樣本分割成為可能，極大降低了分割模型的使用門檻。

應用場景

自動駕駛：需要即時偵測車輛、行人、交通標誌，並進行可行駛區域的語義分割。BEV（鳥瞰圖）感知與 3D 偵測是當前研究熱點。

醫學診斷：腫瘤偵測、器官分割、細胞計數等任務。醫學影像的特殊性在於資料量小、標註需要專業知識、對假陰性容忍度極低。

智慧零售：商品辨識、貨架分析、客流統計。SKU（庫存單位）辨識面臨數千類別與頻繁更新的挑戰。

遙感分析：衛星影像中的建築物偵測、道路提取、土地利用分類。超大解析度影像的處理需要特殊的滑動視窗或分塊策略。

延伸閱讀

  • ImageNet Large Scale Visual Recognition Challenge (ILSVRC) 推動了分類技術的飛速發展
  • COCO（Common Objects in Context）是偵測與分割的標準評測資料集
  • Cityscapes 專注於自動駕駛場景的語義理解
  • 《Object Detection in 20 Years: A Survey》全面回顧偵測技術演進
  • MMDetection 與 Detectron2 提供完整的偵測與分割開源框架

影像理解正朝向統一化、開放世界與多模態方向發展。基礎模型（Foundation Models）的出現讓零樣本與少樣本的視覺理解成為可能，這將深刻改變視覺系統的開發方式。
