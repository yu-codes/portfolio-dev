跨雲身份聯邦

跨雲身份聯邦（Cross-Cloud Identity Federation）是多雲環境中實現統一身份管理的核心機制。它允許使用者透過單一身份在不同雲端供應商的服務之間無縫切換，無需為每個平台維護獨立的帳戶與憑證。這不僅提升了使用者體驗，更大幅降低了身份管理的複雜度與安全風險。

核心概念

身份聯邦的基本原理

身份聯邦建立在信任關係之上。一個身份提供者（Identity Provider, IdP）被多個服務提供者（Service Provider, SP）所信任，使用者只需向 IdP 驗證身份一次，即可存取所有已建立信任關係的服務。

核心術語：

  • Identity Provider（IdP）：負責驗證使用者身份並簽發身份斷言的系統。常見的 IdP 包括 Azure AD（Entra ID）、Okta、AWS IAM Identity Center。
  • Service Provider（SP）：依賴 IdP 進行身份驗證的應用或服務。
  • Federation Trust：IdP 與 SP 之間的信任協議，定義了雙方如何交換身份資訊。
  • Token/Assertion：IdP 簽發的身份證明，包含使用者的身份屬性與權限宣告。

聯邦協定

主要的身份聯邦協定包括：

  1. SAML 2.0（Security Assertion Markup Language）：基於 XML 的企業級聯邦標準，廣泛用於 B2B 場景。支援瀏覽器型 SSO 與屬性傳遞。

  2. OIDC（OpenID Connect）：基於 OAuth 2.0 的現代身份協定，使用 JSON Web Token（JWT）。輕量級、適合雲原生應用與 API 整合。

  3. WS-Federation：Microsoft 生態系統中常用的聯邦標準，與 Active Directory Federation Services（AD FS）緊密整合。

  4. SCIM（System for Cross-domain Identity Management）：專注於身份生命週期管理的標準，用於跨系統同步使用者與群組資訊。

在集中式架構中，企業選擇一個主要的 IdP 作為所有雲端服務的身份來源。所有使用者的身份生命週期（建立、修改、停用）都在中央 IdP 中管理，各雲端透過聯邦信任關係接受中央 IdP 的身份斷言。

關鍵組件

  1. 身份儲存庫（Identity Store）：儲存使用者帳戶、群組與屬性的目錄服務，通常是 LDAP/Active Directory。

  2. 聯邦閘道（Federation Gateway）：處理協定轉換與 Token 交換的中間層，支援不同格式的身份斷言轉換。

  3. 權限對映（Role Mapping）：將中央 IdP 中的群組或屬性對映到各雲端的角色與權限。例如，將 AD 群組「Cloud-Admins」對映到 AWS 的 AdministratorAccess 角色。

  4. 條件式存取（Conditional Access）：基於風險訊號（裝置狀態、位置、行為模式）動態調整存取決策。

Token 交換流程

跨雲存取通常需要 Token 交換機制：

  1. 使用者向中央 IdP 取得初始 Token
  2. 應用程式使用初始 Token 向目標雲端的 Security Token Service（STS）請求交換
  3. STS 驗證初始 Token 並簽發該雲端專用的短期憑證
  4. 應用程式使用短期憑證存取目標雲端資源

應用場景

場景一：企業多雲 SSO

大型企業使用 Azure AD 作為中央 IdP，員工透過 SSO 存取 AWS Console、GCP Console 以及各種 SaaS 應用。IT 團隊在 Azure AD 中管理所有使用者的生命週期，當員工離職時，停用 Azure AD 帳戶即可自動撤銷所有雲端的存取權限。

場景二：Workload Identity Federation

微服務架構中，運行在 GCP 上的服務需要存取 AWS S3 中的資料。透過 Workload Identity Federation，GCP 服務帳戶可以直接交換 AWS 臨時憑證，無需管理長期密鑰。這大幅降低了憑證外洩的風險。

場景三：合作夥伴存取

B2B 場景中，合作夥伴使用自己的 IdP 透過 SAML 聯邦存取企業的雲端資源。企業不需要為合作夥伴建立帳戶，而是信任合作夥伴的 IdP 簽發的身份斷言，並透過屬性對映授予適當的權限。

場景四：開發與生產環境隔離

開發團隊使用不同的 IAM 角色存取開發與生產環境。透過身份聯邦與條件式存取策略，可以確保只有經過額外驗證（如 MFA）且在公司網路內的使用者才能存取生產環境。
