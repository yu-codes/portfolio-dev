Portfolio Website

個人作品集與技術知識庫，使用 Hugo 建構，支援互動式路線圖導航、雙語（中英文）、深色模式，以及自動化雙倉庫部署至 GitHub Pages。

Hugo CSS3 JavaScript Static Site GitHub Pages GitHub Actions
View on GitHub

概述

結合個人作品集、學習中心與專案展示的綜合技術平台。完全使用 Hugo 靜態網站生成器建構，具備精緻的互動式路線圖視覺化系統（XMind 風格）。支援雙語內容（英文與繁體中文），自動無縫部署至 GitHub Pages，無需後端服務。

功能特色

  • XMind 風格互動路線圖：660+ 行 SVG 可摺疊節點系統，含自動佈局演算法、平滑動畫與上下文感知連結渲染，瀏覽 9 大領域 100+ 篇文章
  • 雙倉庫部署：GitHub Actions 工作流程自動將 develop → portfolio-dev、main → portfolio，實現測試/正式環境分離
  • 完整雙語支援：英文與繁體中文獨立內容目錄、語言切換與自動生成導航
  • 精緻深色模式：持久化主題偏好搭配系統偵測回退機制，CSS 變數驅動所有元件主題
  • 模組化 CSS 架構：base.css 設計令牌系統、layout.css 結構佈局、components.css 可重用元件、pages/ 頁面覆寫
  • 行動優先響應式設計：CSS Grid 與 Flexbox 自適應佈局，涵蓋所有斷點
  • SEO 最佳化：自動生成多語言 sitemap、結構化中繼資料、語義化 HTML 與 Open Graph 標籤
