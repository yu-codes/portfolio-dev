Vision Object Detection

基於深度學習的電腦視覺系統，實現即時物件偵測與分類，採用 YOLO 模型、自訂訓練管線，並透過 TorchScript/ONNX 實現生產級部署。

Python PyTorch Computer Vision Deep Learning YOLO
View on GitHub

概述

生產級電腦視覺系統，運用先進深度學習模型實現穩健的物件偵測與分類。本專案實作端到端管線 — 從資料前處理與模型訓練、推論優化到部署 — 應用於設施巡檢與異常偵測等實際場景。

功能特色

  • 即時物件偵測：YOLO v5/v8 實作，RTX 3090 上達 85 FPS、11.8ms 延遲
  • 彈性模型架構：可設定骨幹網路（ResNet、EfficientNet），支援基於錨點與無錨點偵測頭
  • 進階資料增強：完整管線包含旋轉、翻轉、色彩抖動、Mosaic 與 Mixup 增強
  • 遷移學習：基於 COCO 資料集預訓練，支援自訂領域微調
  • 生產匯出：一鍵匯出 TorchScript 與 ONNX 格式，用於邊緣裝置部署
  • 信心度過濾與 NMS：可設定的後處理，支援類別專屬信心度閾值
  • 多格式輸入：支援影像、影片檔案與即時攝影機串流
