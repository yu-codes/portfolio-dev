Agent-Arena

AI Agent 黑盒評測平台 — 評估、量化、比較 LLM Agent 行為表現，內建 12+ 通用指標、客製指標、Plugin 架構與非同步評測引擎。

概述

Agent-Arena（b-box）— Black-Box for Boxing LLM Behaviors — 是一個 AI Agent 黑盒評測平台，用於評估、量化與比較不同 LLM Agent 的行為表現。採用 Plugin 架構實現擴充性，搭配非同步評測引擎平行計算指標，並提供現代化 Vue 3 儀表板與雷達圖視覺化。

功能特色

  • 多 Agent 支援：OpenAI / Anthropic / 自訂 API，可擴充 Adapter 模式 — 實作 BaseAgent 並註冊即可新增
  • 12 項通用指標：語義相似度、事實準確性、幻覺偵測、連貫性、安全性、毒性、回應完整度等
  • 5 項客製指標：依 Agent 類型動態提供 — 工具使用評分、Schema 驗證、精確匹配、格式合規、API 呼叫分析
  • Plugin 架構：新增 Agent 與指標無需修改核心程式碼 — 實作、註冊、部署即可
  • 非同步評測引擎：所有指標使用 Python asyncio 平行執行，最大化吞吐量
  • 報告產出：詳細評測報告可匯出 JSON 與 PDF 格式
  • 現代化儀表板：Vue 3 SPA 搭配雷達圖視覺化、分數明細與即時評測狀態
  • 容器化部署：單一 docker compose up --build 指令啟動全站

API 端點

方法端點說明
