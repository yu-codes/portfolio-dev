Infra-Toolbox

模組化基礎設施服務工具箱 — 每個服務獨立封裝為 Docker Compose 堆疊，支援資源監控、物件儲存、資料庫管理、備份策略與 Claude Code 整合。

概述

Infra-Toolbox 是一套模組化基礎設施服務工具箱，每個服務獨立封裝為 Docker Compose 堆疊。專為後端系統、開發環境與維運工具設計，支援 Windows（Docker Desktop + Git Bash/WSL）與 Linux 環境，使用跨平台 POSIX 相容 Shell 腳本。

目錄結構

infra-toolbox/
├── .github/workflows/ci.yml      # GitHub Actions CI/CD
├── claude-code/                   # Claude Code 開發指南
├── filebrowser/                   # Web 檔案管理器
├── geoserver/                     # GeoServer 地圖服務
├── minio/                         # MinIO S3 相容儲存
├── minio-backup/                  # MinIO 備份服務
├── minio-backup-shell/            # Shell 版 MinIO 備份
├── postgres/                      # PostgreSQL 14（WAL 啟用）
├── postgres-backup-logical/       # 邏輯備份（pg_dump）
├── postgres-backup-physical/      # 物理備份（WAL/PITR）
├── postgres-backup-shell/         # Shell 版 PostgreSQL 備份
├── redis-backup/                  # Redis 備份服務
├── skills/                        # AI Agent Skills 框架
├── system-checks/                 # 系統健康檢查
├── system-monitor/                # 資源監控（CPU/RAM/Storage）
└── tests/                         # 測試框架
    └── run_tests.sh

各服務遵循標準結構：

service_name/
├── docker-compose.yml    # Docker 設定
├── .env.example          # 環境變數範本
├── scripts/              # 自動化腳本（備份/還原）
└── README.md             # 服務文件

功能特色

  • 完全模組化：每個服務獨立部署，執行 docker compose up -d 即可啟動
  • 全方位備份：邏輯備份（pg_dump）、物理備份（WAL/PITR）、加密備份，搭配自動化備份演練測試
  • 跨平台：POSIX Shell 腳本相容 Windows/Linux，自動處理 MSYS_NO_PATHCONV
  • CI/CD 管線：9 階段 GitHub Actions — 單元測試、Docker 驗證、ShellCheck、整合測試、備份演練、加密測試
  • 共享網路：所有服務透過 infra-toolbox-network 通訊
  • 端口規劃：依類別分配（10001-10009 監控、10010-10019 儲存、10020-10029 檔案管理）
  • AI Skills 框架：通用 Agent Skills 格式，相容 Claude、ChatGPT、Cursor 等
