安全性

應用程式安全與防護

全面介紹後端系統資訊安全的四大面向：應用安全、認證安全、基礎設施安全與限流保護，建立從程式碼到基礎設施的縱深防禦體系。

核心概念

資訊安全不是單一技術，而是一套涵蓋設計、開發、部署與運維全生命週期的防禦策略。現代後端系統面臨的威脅日益複雜，從 SQL 注入、XSS 等傳統應用層攻擊，到供應鏈攻擊、API 濫用與分散式阻斷服務（DDoS），攻擊向量不斷演化。

縱深防禦（Defense in Depth） 是安全架構的基本原則。不依賴單一防護層，而是在網路邊界、應用層、資料層和基礎設施層各自建立獨立的安全控制。當某一層被突破時，其他層仍能提供保護。

最小權限原則（Principle of Least Privilege） 要求每個元件只擁有完成其功能所需的最小權限。無論是資料庫連線帳號、API Token 還是容器的檔案系統存取，都應嚴格限制其權限範圍。

零信任架構（Zero Trust） 假設網路內部與外部同樣不可信，每次存取都需要驗證身份與授權。這改變了傳統「城堡與護城河」模型，特別適合微服務和雲端環境。

安全左移（Shift Left Security） 強調在開發流程早期就整合安全實踐，包括安全編碼訓練、靜態分析（SAST）、依賴掃描（SCA）和威脅建模，而非在部署前才進行安全審查。

架構總覽

資訊安全縱深防禦架構
├── 應用安全（Application Security）
│   ├── OWASP Top 10 防護
│   ├── 輸入驗證與輸出編碼
│   ├── SQL 注入防護
│   ├── CSRF/XSS 防護
│   └── 安全標頭配置
├── 認證安全（Authentication Security）
│   ├── 密碼雜湊（bcrypt/Argon2）
│   ├── Token 安全（JWT/Session）
│   ├── 多因子認證（MFA）
│   └── OAuth 2.0 / OIDC
├── 基礎設施安全（Infrastructure Security）
│   ├── 網路隔離與防火牆
│   ├── TLS 憑證管理
│   ├── 機密管理（Vault/KMS）
│   └── 容器安全與映像掃描
└── 限流保護（Rate Limiting & Protection）
    ├── 限流演算法（Token Bucket/Sliding Window）
    ├── DDoS 防護（CDN/WAF）
    ├── API 配額管理
    └── 熔斷與降級策略

這四層形成一個完整的防禦體系。應用安全確保程式碼本身無漏洞；認證安全保護使用者身份不被冒用；基礎設施安全守護運行環境；限流保護則確保服務在異常流量下仍能維持可用性。

關鍵比較

安全面向防護目標典型威脅關鍵技術
應用安全程式碼與邏輯SQL 注入、XSS、SSRF參數化查詢、CSP、輸入驗證
認證安全使用者身份暴力破解、Token 竊取bcrypt、JWT 輪換、MFA
基礎設施安全運行環境網路入侵、機密洩漏mTLS、Vault、網路策略
限流保護服務可用性DDoS、API 濫用Token Bucket、WAF、CDN

被動防禦 vs 主動偵測：傳統防火牆和 WAF 屬於被動防禦，阻擋已知攻擊模式。現代安全實踐還包括主動偵測，如 RASP（Runtime Application Self-Protection）、異常行為分析和蜜罐系統，能發現未知的攻擊行為。

安全性 vs 使用者體驗：過於嚴格的安全控制會損害使用者體驗（如複雜的密碼規則、頻繁的驗證要求）。風險基礎認證（Risk-Based Authentication）根據行為模式動態調整安全等級，在安全與便利之間取得平衡。

應用場景

  1. 金融科技平台：需要最高等級的安全控制，包括 PCI DSS 合規、端對端加密、交易監控與防詐欺系統。認證層需支援生物辨識、硬體安全金鑰等強認證手段。

  2. SaaS 多租戶平台：租戶隔離是核心挑戰，需確保一個租戶的資料不被其他租戶存取。Row-Level Security、獨立加密金鑰和嚴格的 API 授權是基本要求。

  3. 公共 API 平台：面對大量不可控的外部請求，限流保護和 API 金鑰管理至關重要。需要分層配額（免費/付費層級）、請求簽名驗證和異常流量偵測。

  4. 醫療健康系統：受 HIPAA 等法規嚴格約束，需要完整的稽核日誌、資料加密、存取控制和隱私保護機制。資料的生命週期管理和最小化收集原則是設計核心。

  5. 電商平台：需要防護帳號接管（ATO）、信用卡盜用、庫存操控和爬蟲濫用等多種攻擊。限流、設備指紋和行為分析是常用防護手段。

安全是一個持續性的過程而非一次性的任務。定期的滲透測試、安全審計、依賴更新和事件回應演練是維持安全態勢的關鍵活動。
