SQL 資料庫（關聯式資料庫管理系統，RDBMS）是企業級應用的資料儲存基石，自 1970 年代 Edgar Codd 提出關聯模型以來，已歷經五十年的發展與驗證。SQL 資料庫以嚴格的 Schema 定義、強大的查詢語言和 ACID 事務保證著稱，適合需要資料完整性、複雜查詢和多表關聯的業務場景。PostgreSQL 和 MySQL 是當今最流行的開源選擇。

核心概念

關聯模型（Relational Model） 將資料組織為二維表格（Relation），每個表格由固定的欄位（Attribute）和動態的資料列（Tuple）組成。表之間透過主鍵（Primary Key）和外鍵（Foreign Key）建立關係，實現資料的正規化儲存，消除冗餘與更新異常。

SQL 語言分為四大子集：

  • DDL（Data Definition Language）：CREATE、ALTER、DROP — 定義資料庫結構
  • DML（Data Manipulation Language）：SELECT、INSERT、UPDATE、DELETE — 操作資料
  • DCL（Data Control Language）：GRANT、REVOKE — 控制存取權限
  • TCL（Transaction Control Language）：BEGIN、COMMIT、ROLLBACK — 管理事務

事務隔離級別決定並行事務之間的可見性：

  • Read Uncommitted：允許髒讀，幾乎不使用
  • Read Committed：只能讀取已提交資料，PostgreSQL 預設
  • Repeatable Read：同一事務中重複讀取結果一致，MySQL InnoDB 預設
  • Serializable：完全串行化，最高隔離但效能最差

查詢優化器（Query Optimizer） 是 SQL 資料庫的智慧核心，將宣告式的 SQL 查詢轉換為最佳的執行計畫。優化器基於統計資訊（表大小、欄位基數、資料分佈）選擇 JOIN 算法、決定索引使用、安排操作順序。理解 EXPLAIN 輸出是效能調優的基本技能。

架構總覽

SQL 資料庫的內部架構通常分為以下層次：

連接層（Connection Layer）：管理客戶端連接，包括認證、連接池、協議解析。連接池（如 PgBouncer、ProxySQL）能顯著降低連接建立的開銷。

解析與優化層：SQL 解析器將文字轉為抽象語法樹（AST）；語意分析驗證表和欄位的存在性；查詢重寫器進行等價轉換（如子查詢展開）；優化器生成執行計畫。

執行引擎：按照執行計畫驅動各種物理操作（Sequential Scan、Index Scan、Hash Join、Sort 等）。現代引擎支援向量化執行（Vectorized Execution）以提升 CPU 利用率。

儲存引擎：管理資料在磁碟上的組織方式。InnoDB（MySQL）使用聚集索引將資料存放在 B+Tree 葉子節點；PostgreSQL 使用堆表（Heap）加獨立索引。兩者都使用 WAL（Write-Ahead Log）保證事務的持久性。

索引子系統：B+Tree 索引支援等值與範圍查詢；GiST 索引支援地理空間資料；GIN 索引支援全文搜尋與 JSONB；BRIN 索引以極低空間開銷處理自然有序的資料。

應用場景

企業 ERP 系統：訂單、庫存、財務等核心業務模組需要跨表事務保證（如扣庫存同時建立訂單），SQL 資料庫的 ACID 特性確保了業務邏輯的正確性。複雜的報表查詢使用 Window Function 和 CTE（Common Table Expression）高效完成。

SaaS 多租戶架構：透過 Schema-per-tenant 或 Row-Level Security（PostgreSQL RLS）實現租戶隔離。Connection Pooling 支撐大量租戶的並行連接。PostgreSQL 的 JSONB 允許每個租戶有自訂欄位而不需要 Schema 變更。

地理空間應用：PostGIS 擴展為 PostgreSQL 提供完整的地理空間功能，支援點線面幾何計算、空間索引（R-Tree/GiST）、路徑規劃等。外送平台、地圖服務、城市規劃系統大量使用空間查詢。

金融交易處理：銀行轉帳必須在一個事務內完成扣款與入帳；序列化隔離防止超賣（Oversell）；樂觀鎖（Optimistic Locking）提升高併發場景的吞吐量；審計表結合觸發器記錄所有資料變更歷史。

數據分析與 BI：PostgreSQL 支援複雜的分析查詢（Window Functions、遞迴 CTE、GROUPING SETS）；物化視圖（Materialized View）加速重複的聚合計算；COPY 命令高效批量載入資料。
