請求生命週期

深入理解一個 HTTP 請求從用戶端到後端的完整生命週期，涵蓋每個階段的可觀測性切入點

理解一個請求的完整生命週期是後端工程師的核心能力。從使用者點擊按鈕到收到回應，一個請求會經過 DNS 解析、網路傳輸、負載均衡、應用處理、資料存取等多個階段。掌握每個階段的行為和可觀測性切入點，才能有效地診斷和優化系統。

關鍵比較

同步 vs 非同步請求處理

維度同步處理非同步處理
使用者等待等到完成立即回應確認
延遲影響直接感受不影響回應時間
錯誤處理同步回傳需補償機制
追蹤挑戰簡單（一條線）複雜（分叉、延遲關聯）
適用場景查詢、即時操作寫入、通知、背景處理
