機器學習的數學與統計基礎

機器學習的基礎建立在數學與統計學之上。線性代數提供了資料與模型的表示語言，機率統計賦予我們量化不確定性的工具，而微積分（特別是梯度）則驅動了模型的學習過程。深入理解這些基礎概念，是進階到複雜演算法與系統設計的必要前提。

核心概念

線性代數基礎

線性代數是機器學習的骨幹語言：

  • 向量與矩陣：資料以向量（單筆樣本）或矩陣（批次資料）形式存在，模型參數同樣以矩陣表示
  • 矩陣乘法：神經網路的前向傳播本質上是一連串矩陣乘法加上非線性變換
  • 特徵分解與 SVD：用於降維（PCA）、推薦系統（矩陣分解）、資料壓縮
  • 範數（Norm）：$L_1$ 範數促進稀疏性，$L_2$ 範數控制權重大小，是正則化的基礎

機率與統計

概念在 ML 中的角色
貝氏定理後驗推論、貝氏最佳化、不確定性估計
最大似然估計（MLE）許多損失函數的理論根基
期望值與變異數偏差-變異數權衡的基礎
分佈假設生成模型、異常偵測的前提
假設檢定A/B 測試、模型比較的統計顯著性
資訊理論交叉熵損失、KL 散度、互資訊

損失函數

損失函數將模型的預測品質量化為單一數值，引導優化方向：

分類損失：

  • 交叉熵（Cross-Entropy）：$L = -\sum_{i} y_i \log(\hat{y}_i)$
  • Hinge Loss：SVM 的核心，鼓勵正確類別的分數遠大於錯誤類別
  • Focal Loss：針對類別不平衡，降低易分類樣本的權重

迴歸損失：

  • MSE：$L = \frac{1}{n}\sum(y - \hat{y})^2$，對離群值敏感
  • Huber Loss：在殘差小時為 MSE，大時為 MAE，兼具兩者優點
  • Quantile Loss：預測條件分位數而非均值

梯度下降

梯度下降是幾乎所有深度學習訓練的核心：

# 基本梯度下降更新規則
theta = theta - learning_rate * gradient(loss, theta)

主要變體包括：

  • 批次梯度下降（BGD）：使用全部資料計算梯度，穩定但慢
  • 隨機梯度下降（SGD）：每次用一筆資料，快但雜訊大
  • 小批次梯度下降（Mini-batch SGD）：折衷方案，實務標準
  • 動量法（Momentum）：累積歷史梯度方向，加速收斂
  • Adam：自適應學習率，結合動量與 RMSProp 的優點

架構總覽

ML 基礎知識的架構層次：

數學層：線性代數（向量空間、矩陣運算）、微積分（偏導數、鏈式法則）、機率論（分佈、條件機率）。這些構成了所有 ML 演算法的理論基礎。

演算法層：傳統機器學習演算法（線性迴歸、邏輯迴歸、決策樹、SVM、k-NN）利用數學工具解決特定問題模式。每個演算法都有其假設、優勢與適用場景。

最佳化層：梯度下降及其變體、凸最佳化、約束最佳化。這一層決定了模型如何從資料中學習。

評估層：偏差-變異數分解、學習曲線分析、統計檢定。確保模型學到的是真實模式而非雜訊。

關鍵比較

傳統 ML vs 深度學習

面向傳統 ML深度學習
特徵工程手動設計，需領域知識自動學習表示
資料需求小資料即可表現佳需大量資料
可解釋性通常較高黑箱，需額外工具
計算資源CPU 即可通常需 GPU
理論基礎統計學習理論經驗主導

生成模型 vs 判別模型

生成模型（如 Naive Bayes、GMM）學習聯合分佈 $P(X, Y)$，能生成新樣本且對缺失資料魯棒。判別模型（如 Logistic Regression、SVM）直接學習條件分佈 $P(Y|X)$，通常在分類任務上更準確。

參數模型 vs 非參數模型

參數模型（線性迴歸）假設固定的函數形式，模型複雜度不隨資料增長。非參數模型（k-NN、決策樹）讓資料決定模型複雜度，更靈活但可能過擬合。

應用場景

  1. 線性迴歸分析：房價預測、銷售預測等連續值任務的起點模型
  2. 邏輯迴歸分類：信用評分、疾病篩檢等二元分類的基線方法
  3. 決策樹與整合方法：Random Forest、XGBoost 在結構化資料上的卓越表現
  4. SVM 核方法：小樣本高維問題的有力工具
  5. 貝氏推論：不確定性估計、超參數優化的理論框架
  6. 降維與視覺化：PCA、t-SNE 用於資料探索與特徵壓縮
