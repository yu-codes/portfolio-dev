Nuxt 是基於 Vue.js 的全端框架，以「慣例優於配置」為哲學，提供自動匯入、檔案路由、多種渲染模式與強大的模組系統。Nuxt 3 搭載全新伺服器引擎 Nitro，支援多平台部署，並充分利用 Vue 3 的 Composition API 與 TypeScript 支援。

應用場景

內容驅動網站

搭配 @nuxt/content 模組，可直接以 Markdown 檔案管理內容，支援 MDC（Markdown Components）語法嵌入 Vue 元件。搭配 Nuxt Studio 視覺化編輯器，讓非技術人員也能管理內容。適合文件網站、技術部落格與行銷頁面。

多地區電商

Nuxt 的混合渲染策略搭配 i18n 模組，可為不同地區、不同頁面配置最佳渲染模式。商品列表使用 ISR、個人化推薦使用 SSR、靜態內容使用 SSG，最大化效能與 SEO 表現。

企業級 SaaS

自動匯入與模組系統大幅降低大型專案的設定複雜度。Layers 功能允許多個 Nuxt 專案共享配置、元件與 composable，適合微前端架構或白標解決方案。
