泛型（Generics）是 TypeScript 型別系統中最強大的抽象工具，它允許我們撰寫可重用且型別安全的程式碼。透過參數化型別，泛型在保持彈性的同時不犧牲型別檢查的嚴謹性。

核心概念

泛型的基本元素

元素語法說明範例
型別參數<T>代表待定的型別function id<T>(x: T): T
型別約束<T extends U>限制型別參數範圍<T extends object>
預設型別<T = Default>未指定時的預設值<T = string>
多型別參數<T, U, V>多個獨立的型別參數<K, V> 用於 Map
推斷自動從引數推斷型別參數id(42) → T = number

泛型的應用位置

位置語法範例說明
函式function f<T>(x: T): T最常見的泛型使用
介面interface Box<T> { value: T }泛型資料結構
類別class Stack<T> { ... }泛型容器
型別別名type Result<T> = Success<T> | Error泛型工具型別
方法method<U>(x: U): void類別/介面內的泛型方法

型別約束（Constraints）

約束形式語法作用
extends<T extends object>T 必須是物件
keyof<K extends keyof T>K 必須是 T 的鍵
聯合約束<T extends string | number>T 限於指定型別
建構函式約束<T extends new() => any>T 必須可 new
遞歸約束<T extends Comparable<T>>T 必須實作比較

架構總覽

泛型推斷機制

呼叫 function pick<T, K extends keyof T>(obj: T, keys: K[]): Pick<T, K>

步驟：
1. 從第一個引數推斷 T → { name: string; age: number }
2. 從 K extends keyof T 得知 K ⊂ "name" | "age"
3. 從第二個引數推斷 K → "name"
4. 回傳型別計算為 Pick<{ name: string; age: number }, "name">
5. 最終結果：{ name: string }

協變與逆變在泛型中的表現

位置泛型行為範例
type Producer<T> = () => T協變Producer<Dog> → Producer<Animal> ✓
type Consumer<T> = (x: T) => void逆變Consumer<Animal> → Consumer<Dog> ✓
type Both<T> = (x: T) => T不變不可互相賦值
type ReadonlyBox<T> = { readonly value: T }協變安全的子型別關係

條件型別中的泛型

條件型別（Conditional Types）讓泛型具備邏輯推理能力：

T extends U ? X : Y

分配律（Distributive）：當 T 是聯合型別時自動分配
(A | B) extends U ? X : Y
→ (A extends U ? X : Y) | (B extends U ? X : Y)

泛型約束策略

策略語法效果適用場景
無約束<T>任意型別真正通用的容器
結構約束<T extends { id: string }>必須有特定結構需要存取特定屬性
鍵約束<K extends keyof T>限於物件的鍵型別安全的屬性存取
映射約束<T extends Record<string, any>>必須是記錄型別通用物件操作
函式約束<F extends (...args: any[]) => any>必須是函式高階函式

常見泛型模式

模式說明範例
容器型別包裝任意型別的值Box<T>, Promise<T>
工廠模式建立泛型實例create<T>(ctor: new() => T): T
轉換函式保留型別的映射map<T, U>(arr: T[], fn: (x: T) => U): U[]

型別安全事件事件名與處理器對應on<K extends keyof Events>(event: K, handler: (data: Events[K]) => void)

進階技巧

infer 關鍵字

在條件型別中提取子型別：

type ReturnType<T> = T extends (...args: any[]) => infer R ? R : never;
type ElementType<T> = T extends (infer U)[] ? U : never;
type PromiseValue<T> = T extends Promise<infer V> ? V : T;

遞歸泛型

type DeepReadonly<T> = {
  readonly [K in keyof T]: T[K] extends object ? DeepReadonly<T[K]> : T[K];
};
