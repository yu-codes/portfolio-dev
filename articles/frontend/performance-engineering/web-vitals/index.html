Web Vitals 是 Google 定義的一組量化 Web 使用者體驗的關鍵指標，其中 Core Web Vitals（核心網頁指標）更直接影響 SEO 排名。理解這些指標的含義、測量方式與優化路徑，是前端效能工程的必備知識。

應用場景

持續效能監控

整合 web-vitals 庫至應用中，收集真實使用者的 LCP、INP、CLS 資料上報至分析平台。設定效能預算（Performance Budget），指標退化時自動告警。搭配 CI 中的 Lighthouse 檢查，防止效能退化進入生產。

電商效能優化

電商網站的 Core Web Vitals 直接影響轉換率。研究顯示：LCP 每改善 100ms，轉換率提升 1-2%。優化路徑：Hero 圖片 preload + WebP、產品列表虛擬化、廣告位預留空間（CLS）、互動元素減少 JS 阻塞（INP）。

第三方腳本管理

分析、廣告、聊天等第三方腳本常是效能殺手。策略：使用 Partytown 將第三方 JS 移至 Web Worker；使用 loading="lazy" 延遲非關鍵 iframe；監控各第三方腳本的 TBT 貢獻，移除或替換影響過大的服務。
